### Requirements
- JDK 8 or higher
- Spring Boot (v.2.7.4)
- Docker

### Docker instructions
- Build the image from the Dockerfile
```
docker build -t <image name> <path to Dockerfile>
```
- Run the application from the image in the Dockerfile directory
```
docker run -p<your port>:8080 <image name>
```
